# Social Training Module

## 목적

발달장애인이 직장 내에서 타인과 관계를 맺고, 필요한 말을 적절히 하며, 갈등이나 실수 상황에서도 안정적으로 대응할 수 있도록 지원한다.

## 구성 방식

- 실시간 음성 기반 AI 대화형 훈련
- AI가 상사, 동료, 담당자 등의 역할로 말을 걸고, 사용자가 직접 말로 응답하는 방식

## 주요 내용

- 부당한 지시를 받았을 때 적절히 반응하기
- 실수했을 때 사과하고 상황을 정리하기
- 이해가 되지 않을 때 도움을 요청하기
- 상사와 동료에게 적절한 방식으로 말 걸기
- 직장 내 기본적인 인사와 응답 익히기
